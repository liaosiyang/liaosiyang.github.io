Java-内存区域与内存溢出异常

2016-08-03
  • Java
  • JVM

JVM是在千差万别的物理机上建立了统一的运行平台，实现了在任意一台虚拟机上编译的程序都能在任何一台虚拟机上正常运行。这让开发人员可以把主要精力集中在具体的业务逻辑上，而不是物理硬件的兼容性上。
这么说来，开发人员只要了解Java API、Java语法就行了，就可以满足日常开发需要了。(当然不包括高级的开发)
学习JVM首先是Android开发是基于Java的，所以想要了解Java到底是如何运行起来的，有什么运行机制和原理。二是如果开发人员不了解虚拟机一些技术特性的运行原理，就无法写出最适合虚拟机运行和自优化的代码。
Java虚拟机首先实现了自动内存管理机制，像一个管家一样，把它管理的内存划分为若干个不同的数据区域。有些区域还会出现爆满的情况，这是就会抛出异常。
总结一下Java虚拟机将内存划分的几个区域、对象的创建和内存溢出异常。

第一次就随便写写吧

2016-07-31
  • 其它
  • other

花了2天时间，github pagers+hexo搭建的个人主页基本初步完成了。主题来自github.com/denjones/hexo-theme-chan

域名的话有时间买一个吧，主题的话还不是很满意，还需调整，评论什么的也没做，最近也比较忙，就先这样吧。

为什么要建立这个小站，说实在起因是一时兴起，我平时也并没有写日记、随笔的习惯。技术方面略有总结，基本都写在为知笔记里了。那些总结实在也没什么拿出来分享的必要，都是别人造的轮子。

既然做了这个小站，目的就是为了鼓励自己在技术方面有所突破，让自己有拿得出手的东西。看懂别人的文章和自己总结不是一回事，自己总结和分享给他人更不是一回事。

马上就要校招了，之后就要毕业了。在此先祝愿自己能找到一个技术氛围好的团队，既然选择了这一行业，那就好好努力地干下去吧。
